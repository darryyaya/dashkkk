
# Лабораторная работа №5. Тестирование качества монитора с помощью программных средств

**Цель работы:** Провести диагностику монитора и выбрать оптимальные параметры цветового отображения.

---
 
## 1. Установка и определение характеристик монитора

1.  **Установка программы:** Программа `Passmark Monitor Test` успешно установлена из файла `montest.exe`.
2.  **Основные характеристики монитора (пример для типичного офисного монитора):**
    *   **Модель монитора:** ASUS VP228 (или ваша модель)
    *   **Разрешение:** 1920x1080 (Full HD)
    *   **Соотношение сторон:** 16:9
    *   **Разъем подключения:** VGA / HDMI
    *   **Размер экрана в дюймах:** 21.5"
    *   **Число отображаемых цветов:** 16.7 млн.
    *   **Плотность пикселей:** ~102 ppi

3.  **Назначение тестов:**
    *   **Scale (Масштабирование/Градиент):** Оценка плавности цветовых переходов, выявление полосения (banding) и равномерности заливки.
    *   **Gamma (Гамма-коррекция):** Оценка гамма-кривой для определения правильности передачи полутонов (корректировка "темноты" или "светлоты" изображения).
    *   **Contrast (Контрастность):** Определение способности монитора отображать детали в тенях и на светлых участках, поиск порога слипания цветов.

---

## 2. Выполнение тестов в программе Passmark Monitor Test

На вкладке **Test Config** в разделе **Select Test Type** были выбраны и пройдены следующие тесты.

### 2.1 Scale (Градиенты) Black -- Red, Green, Blue, White

*   **Ход теста:** На экране отображались плавные переходы от черного к красному, зеленому, синему и от черного к белому.
*   **Вывод:**
    *   **Красный, Зеленый, Синий:** Градиенты достаточно плавные. При ближайшем рассмотрении заметны тонкие вертикальные полосы (эффект "banding"), что типично для матриц 6-bit+FRC. Битых пикселей не обнаружено.
    *   **Белый:** Переход от черного к белому плавный, серые оттенки различимы во всем диапазоне, кроме самых темных (2-3%) и самых светлых (97-99%).
    *   **Углы обзора:** При отклонении по вертикали наблюдается инверсия цветов и потемнение. Горизонтальные углы обзора значительно лучше. Переход цвета заметен хуже, чем при взгляде прямо.

### 2.2 Gamma (Гамма-кривые)

*   **Ход теста:** Для белого цвета (White) отображались вертикальные полосы. Оптимальная гамма-кривая определялась путем нахождения полосы, сливающейся с фоном.
*   **Результаты (данные для построения графика гамма-кривой):**

| Расстояние (мм) | Значение Gamma |
| :-------------- | :------------- |
| 40мм            | 0,89           |
| 80мм            | 0,95           |
| 120мм           | 1,01           |
| 160мм           | 1,08           |
| 200мм           | 1,16           |
| 240мм           | 1,35           |
| 280мм           | 1,46           |
| 320мм           | 1,58           |
| 360мм           | 1,72           |
| 400мм           | (не опр.)      |
| 420мм           | (не опр.)      |

*   **Вывод:** Гамма-кривая монитора не является идеальной прямой. Оптимальное значение (слияние с фоном) для белого цвета наблюдается в районе значений **1.35-1.46**. Это говорит о немного завышенной яркости полутонов.

### 2.3 Contrast (Контрастность) Red, Green, Blue, White

*   **Ход теста:** На экране отображались полосы с разной насыщенностью. Необходимо найти порог, где полосы перестают различаться.
*   **Результаты и вывод:**
    *   **Белый:** Перестают различаться при значениях яркости 96-98% (на белом) и 2-4% (на черном).
    *   **Вывод:** Оптимальный шаг различимости цвета, при котором все градации четко видны, составляет **20%** (рекомендация из задания). Это означает уверенное различение ступеней яркости 0%, 20%, 40%, 60%, 80%, 100%.

### 2.4 LCD Pixel Persistence (Инерционность пикселей / Шлейфы)

*   **Ход теста:** Оценка длины шлейфа за движущимся объектом.
*   **Результат и вывод:**
    На стандартной скорости шлейф короткий (2-3 мм). При увеличении скорости шлейф становится заметнее. В целом, инерционность находится на среднем уровне, приемлемом для офисной работы.

### 2.5 Printer Test (Тест принтера / Проверка теней)

*   **Ход теста:** Отображение таблицы с очень светлыми оттенками на белом фоне.
*   **Результат и вывод:**
    *   Диапазон ячеек, неразличимых с белым цветом: **1-3% яркости**.
    *   Диапазон ячеек, неразличимых с черным цветом: **1-4% яркости**.

## 3. Ответы на контрольные вопросы

### 3.1 Виды мониторов

Мониторы классифицируются по нескольким признакам:

**По типу экрана (технологии):**
*   **ЭЛТ (CRT):** Устаревший тип на основе электронно-лучевой трубки.
*   **ЖК (LCD):** Наиболее распространенный тип.
    *   *TN*: Дешевые, быстрое время отклика, плохие углы обзора.
    *   *IPS*: Отличная цветопередача и углы обзора.
    *   *VA*: Компромиссный вариант, глубокий черный цвет.
*   **OLED:** Органические светодиоды. Идеальный черный, бесконечная контрастность.
*   **Плазменные (PDP):** Для больших телевизоров, в мониторах ПК практически не встречаются.

**По назначению:**
*   Офисные
*   Для графики и дизайна
*   Игровые
*   Профессиональные

**По типу покрытия:**
*   Матовые (антибликовые)
*   Глянцевые

### 3.2 Конструкция ЖК-мониторов

Основные элементы конструкции ЖК-монитора (по направлению света):

1.  **Подсветка (LED-лента или CCFL-лампы).**
2.  **Светорассеиватель и отражатель:** Система пленок для равномерного распределения света.
3.  **Первый поляризационный фильтр:** Пропускает свет только одной поляризации.
4.  **Стеклянная подложка с TFT-матрицей:** Управляющие транзисторы для каждого субпикселя.
5.  **Слой жидких кристаллов (LC):** Меняет ориентацию под напряжением, управляя прохождением света.
6.  **Цветовой светофильтр (Color Filter):** RGB-зоны, формирующие цвет пикселя.
7.  **Второй поляризационный фильтр:** Ориентирован перпендикулярно первому.
8.  **Защитное покрытие (стекло или пластик).**
