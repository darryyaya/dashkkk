# Отчёт по практической работе №8

## Тема: Диагностика и оценка производительности ПК с использованием Passmark PerformanceTest

**Цель занятия:** Выполнить комплексную проверку стабильности работы компьютерной системы и на основе полученных результатов сформулировать предложения для составления технического задания на внедрение оборудования.

## Краткие теоретические основы

Утилита **PerformanceTest** представляет собой пакет эталонных тестов (бенчмарков) для измерения быстродействия различных подсистем ПК. Программа позволяет:

- Определить, соответствует ли производительность компьютера ожидаемому уровню;
- Сопоставить результаты с показателями аналогичных или более мощных конфигураций;
- Получить объективные данные для принятия решений о покупке или модернизации;
- Создавать собственные сценарии тестирования.

В составе PerformanceTest имеются следующие группы тестов:

| Группа | Что оценивается |
|--------|----------------|
| CPU | Математические вычисления, шифрование, работа с MMX/SSE/3DNow! |
| 2D графика | Рисование примитивов, работа с битмапами, шрифтами, GUI |
| 3D графика | DirectX 8.1/9, анимация |
| Диск | Линейное чтение/запись, время произвольного доступа |
| Память | Скорость выделения и доступа, пропускная способность |
| CD/DVD | Скорость чтения оптических носителей |

Платная версия также включает расширенные тесты для дисков, сети, памяти и режима многозадачности.

### Ответы на вопросы

#### 1. Где можно применить данный компьютер?

Конфигурация (i3-2120, 4


ГБ, SSD 120 ГБ) подойдёт для:

- Базового офисного пакета (Word, Excel, почта);
- Обучения азам компьютерной грамотности;
- Роли терминального клиента для подключения к удалённому рабочему столу;
- Простого файлового сервера или сервера печати.

**Не подходит для:** Windows 11, современных игр, видеомонтажа, работы с большим количеством вкладок в браузере.

#### 2. Что можно улучшить и с какой выгодой?

| Параметр | Сейчас | Рекомендация | Приблизительная цена (б/у) |
|----------|--------|--------------|----------------------------|
| ОЗУ | 4 ГБ | Добавить ещё 4 ГБ (всего 8 ГБ) | 1500–2000 руб. |
| Диск | 120 ГБ | Заменить на 240–480 ГБ | 1500–2500 руб. |
| Процессор | i3-2120 | Поставить i7-2600 (тот же сокет LGA1155) | 2500–3500 руб. |

**Самое эффективное вложение:** увеличение ОЗУ до 8 ГБ — сразу станет легче работать с браузером и офисными программами.
